# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-03-05

### Added

- Initial release of @arylo/esbuild-plugin-css-minify-text
- Support for minifying `.css` files using esbuild transform
- Configurable minification option via `enableMinify` parameter
- Integration with esbuild as a text loader plugin
- TypeScript support with full type definitions
